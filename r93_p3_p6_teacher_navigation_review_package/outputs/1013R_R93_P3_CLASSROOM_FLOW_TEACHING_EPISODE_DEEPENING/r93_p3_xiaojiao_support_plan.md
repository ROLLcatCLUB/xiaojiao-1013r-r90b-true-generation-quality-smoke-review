# 小教环节支持计划

小教在本轮只作为教师审核草案中的辅助角色，不接真实 UI，不写库。

## 看见渐变

- 提示卡：如果学生只说“好看”，提示他说清起点和终点。
- 追问卡：它中间有没有一层一层的变化？
- 误区提醒：学生可能把颜色多误认为渐变。
- 证据记录建议：记录能说出“从___到___”的学生。

## 分清亮暗与鲜灰

- 提示卡：先问学生看的是亮不亮，还是鲜不鲜。
- 追问卡：如果只能选一个词，你会选亮、暗、鲜、灰里的哪一个？
- 误区提醒：学生可能把变浅和变灰混为一谈。
- 证据记录建议：记录能完成分类并说出理由的学生。

## 三格试色

- 提示卡：提醒学生先做三格，不急着画完整作品。
- 追问卡：中间这一格能不能再靠近第一格一点？
- 误区提醒：学生可能把三种不同颜色并排当成渐变。
- 证据记录建议：保留三格小样作为过程证据。

## 放进作品

- 提示卡：如果学生画面太复杂，提醒先保证一条清楚的渐变。
- 追问卡：这幅作品里最清楚的一段渐变在哪里？
- 误区提醒：学生可能追求复杂图案，忽略渐变规律。
- 证据记录建议：记录作品中能看出明确变化方向的例子。

## 自查与微修订

- 提示卡：提醒教师把评价压缩为一处微修，不展开大评比。
- 追问卡：这一处改完以后，起点和终点更清楚了吗？
- 误区提醒：学生可能把评价理解为重新画一遍。
- 证据记录建议：保留修改前后局部照片或学习单勾选。
