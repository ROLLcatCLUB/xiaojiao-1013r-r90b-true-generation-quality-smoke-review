0.1 这节课的主线

  • 先让学生看见颜色慢慢变化。
  • 再把变化分成亮暗与鲜灰两类。
  • 接着通过三格试色把规律做出来。
  • 然后把试色规律迁移到作品。
  • 最后用自查和微修订把评价落到作品改进上。

第1个环节 · 约5分钟

1看见渐变

环节目标：学生能从生活图或教材图中指出颜色从哪里开始、变到哪里。

1.1 老师先做

出示图片，先请学生安静观察，再邀请学生用手指出颜色变化方向。

先不急着说专业词。你只要找到：颜色从哪里开始，慢慢变到了哪里。

1.2 学生跟着做

先看：观察图片中的颜色层次。

再做：用手指从颜色起点滑到终点。

表达：我看到颜色从___变到___。

1.3 这一环节看什么

学生能圈出或指出起点与终点。

学生能说出一句“从___到___”。

展开 1.4-1.9 支架、大屏、追问和设计意图

1.4 大屏与视觉材料

天空、山峦、花朵或教材页局部三图并排，每张图只保留一个箭头。

  • 找一找：颜色从哪里变到哪里？
  • 用手指一指变化方向。
  • 圈出起点和终点，用箭头标出变化方向。

1.5 教师追问

  • 你是从哪里看出它在慢慢变的？
  • 它是一下一下跳过去，还是一层一层变过去？

1.6 学生支架

  • 从___到___
  • 颜色慢慢变过去
  • 图片箭头、起点终点标记。
  • 同桌先互相指一处，再全班分享。

1.7 小教支持

  • 如果学生只说“好看”，提示他说清起点和终点。
  • 它中间有没有一层一层的变化？
  • 学生可能把颜色多误认为渐变。
  • 记录能说出“从___到___”的学生。

1.8 误区与补救

  • 把颜色丰富等同于渐变。
  • 只说好看，不说变化方向。
  • 减少图片数量，只保留一张最清楚的图。
  • 让学生先用手指，不急着口头表达。

1.9 设计意图

先用直观观察建立“慢慢变”的经验，避免一上来讲明度、纯度造成负担。

过渡：既然能看见颜色慢慢变，接下来要分一分：它是在变亮变暗，还是变鲜变灰。

第2个环节 · 约7分钟

2分清亮暗与鲜灰

环节目标：学生能用儿童化语言初步区分亮暗变化和鲜灰变化。

2.1 老师先做

先指左侧色条问亮暗，再指右侧色条问鲜灰，最后组织分类。

我们先不用急着记词。你只要先看出来：它是变亮了，还是变灰了？

2.2 学生跟着做

先看：观察两组色条。

再做：把色条或色卡放到“亮暗”或“鲜灰”类别。

表达：这一组颜色从___变得更___。

2.3 这一环节看什么

学生能把色条放入正确类别。

学生能用“越来越___”说明变化。

展开 2.4-2.9 支架、大屏、追问和设计意图

2.4 大屏与视觉材料

左侧一组由深到浅的色条，右侧一组由鲜到灰的色条。

  • 亮一点 / 暗一点
  • 鲜一点 / 灰一点
  • 板书两列：亮暗；鲜灰。教师可在旁边小字写明度、纯度。

2.5 教师追问

  • 你看的是颜色亮不亮，还是鲜不鲜？
  • 这组颜色越来越亮，还是越来越灰？

2.6 学生支架

  • 越来越亮
  • 越来越暗
  • 越来越鲜
  • 越来越灰
  • 两组对比色条、分类卡。
  • 同桌先判断，再举卡表达。

2.7 小教支持

  • 先问学生看的是亮不亮，还是鲜不鲜。
  • 如果只能选一个词，你会选亮、暗、鲜、灰里的哪一个？
  • 学生可能把变浅和变灰混为一谈。
  • 记录能完成分类并说出理由的学生。

2.8 误区与补救

  • 把颜色变浅都说成变灰。
  • 只记明度、纯度词语，不会判断。
  • 暂时不要求说专业词，只要求分到亮暗或鲜灰。
  • 用更夸张的两组色条降低判断难度。

2.9 设计意图

教材页同时出现明度与纯度，先用“亮暗、鲜灰”建立直观判断，再轻轻对应专业词。

过渡：能看出来还不够，下一步我们试着自己做出三格慢慢变化。

第3个环节 · 约8分钟

3三格试色

环节目标：学生能完成三格或五格连续变化小样，知道每次只变一点点。

3.1 老师先做

现场示范三格小样，展示一个自然例和一个跳色反例。

先把颜色排好队，再放进你的作品里。第二格不要跑太远，它要像第一格的朋友。

3.2 学生跟着做

先看：观察教师示范和反例。

再做：选择亮暗或鲜灰路线，完成三格小样。

表达：我的颜色从___变到___，中间越来越___。

3.3 这一环节看什么

学生完成三格小样。

小样能看出连续变化。

展开 3.4-3.9 支架、大屏、追问和设计意图

3.4 大屏与视觉材料

三格小样示范：第一格原色，第二格轻微变化，第三格继续变化。

  • 每次只变一点点
  • 第1格 -> 第2格 -> 第3格
  • 标出变化箭头，圈出跳色太快的反例。

3.5 教师追问

  • 第二格更像第一格，还是一下子跳到第三格？
  • 如果太突然，你准备把哪一格改一改？

3.6 学生支架

  • 第一格是___
  • 第二格更___
  • 第三格更___
  • 三格空框、小样纸、色卡或颜料。
  • 同桌帮看中间有没有跳太快。

3.7 小教支持

  • 提醒学生先做三格，不急着画完整作品。
  • 中间这一格能不能再靠近第一格一点？
  • 学生可能把三种不同颜色并排当成渐变。
  • 保留三格小样作为过程证据。

3.8 误区与补救

  • 颜色跳太快。
  • 只换颜色，不形成连续变化。
  • 水分或力度不均导致颜色脏。
  • 材料复杂时改用彩铅、油画棒或色卡排列。
  • 只要求三格，不要求五格。

3.9 设计意图

把概念转成可操作的小任务，降低完整创作前的风险。

过渡：小样做好了，接下来把这条变化规律放进你的图形或小作品里。

第4个环节 · 约14分钟

4放进作品

环节目标：学生能把试色规律迁移到色条、图形或小作品中。

4.1 老师先做

发布分层任务，巡视时只抓起点、方向、过渡三个点。

你不需要画很复杂。今天最重要的是让别人看出颜色是怎样慢慢变过去的。

4.2 学生跟着做

先看：看任务选择和示例。

再做：选择基础、进阶或挑战任务完成作品。

表达：我把渐变放在___，颜色从___变到___。

4.3 这一环节看什么

作品或局部设计中有明确渐变方向。

学生能说出渐变放在哪里。

展开 4.4-4.9 支架、大屏、追问和设计意图

4.4 大屏与视觉材料

展示三类任务：基础色条、进阶图形、挑战小作品。

  • 基础：三格色条
  • 进阶：放进图形
  • 挑战：放进小作品
  • 提示起点、方向、中间过渡三个检查点。

4.5 教师追问

  • 你的颜色从哪里开始变？
  • 别人一眼能看出变化方向吗？
  • 你最想让哪一处颜色变得更自然？

4.6 学生支架

  • 我选择___任务
  • 我的变化方向是___
  • 色条模板、图形轮廓、小样纸、学习单。
  • 同桌先看方向是否清楚。

4.7 小教支持

  • 如果学生画面太复杂，提醒先保证一条清楚的渐变。
  • 这幅作品里最清楚的一段渐变在哪里？
  • 学生可能追求复杂图案，忽略渐变规律。
  • 记录作品中能看出明确变化方向的例子。

4.8 误区与补救

  • 图案太复杂，渐变看不出来。
  • 只涂满颜色，没有方向。
  • 时间不足时只完成一个色条或局部。
  • 让学生把小样直接贴入作品作为局部设计。

4.9 设计意图

让渐变从练习变成艺术表现，避免停留在色卡层面。

过渡：作品有了变化方向，最后我们用自查表看一看：哪里已经清楚，哪里还可以微修。

第5个环节 · 约6分钟

5自查与微修订

环节目标：学生能依据自查或同伴建议，完成一处小修改并说清变化。

5.1 老师先做

组织学生用三项自查看作品，再请同桌给一条可修改建议。

今天不比谁画得最多。我们只看：你能不能让一处颜色变化更清楚。

5.2 学生跟着做

先看：对照自查项看自己的小样或作品。

再做：选择一处进行微修订。

表达：我改了___，因为这里原来___，现在___。

5.3 这一环节看什么

学生完成自查或同伴建议。

作品中有一处可见微修。

展开 5.4-5.9 支架、大屏、追问和设计意图

5.4 大屏与视觉材料

一张作品局部，旁边显示修改前后小箭头。

  • 看得出方向吗？
  • 中间跳太快了吗？
  • 我能改一处吗？
  • 只改一处即可，不要求重画。

5.5 教师追问

  • 你准备改哪一处？
  • 改完以后，颜色变化会更清楚吗？

5.6 学生支架

  • 我发现___
  • 我准备改___
  • 改完以后___
  • 学生自评三项、同伴建议句式。
  • 同伴只提一处建议，避免学生重画。

5.7 小教支持

  • 提醒教师把评价压缩为一处微修，不展开大评比。
  • 这一处改完以后，起点和终点更清楚了吗？
  • 学生可能把评价理解为重新画一遍。
  • 保留修改前后局部照片或学习单勾选。

5.8 误区与补救

  • 评价变成只说好不好看。
  • 学生想重画导致时间不够。
  • 时间不足时只做口头说明，不强制完成完整修改。
  • 只选2-3件作品全班看一处修改。

5.9 设计意图

把评价落到作品改进上，而不是只停留在展示和表扬。

过渡：课后可以继续在生活中找渐变，下节课再看渐变怎样形成节奏。
