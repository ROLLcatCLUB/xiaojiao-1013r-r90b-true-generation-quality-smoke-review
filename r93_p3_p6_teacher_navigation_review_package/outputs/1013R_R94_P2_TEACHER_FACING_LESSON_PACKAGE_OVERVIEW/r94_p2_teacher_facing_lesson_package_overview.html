教学设计摘要

核心问题

颜色怎样慢慢变化，才能让画面更有秩序、更有层次？

学生要做到

  • 能发现生活和教材图中的渐变。
  • 能区分亮暗变化与鲜灰变化。
  • 能做出三格或五格颜色慢慢变化的小样。
  • 能把渐变用到作品里，并说清颜色怎么变。

课堂主线

  • 观察生活和教材图中的颜色变化。
  • 用学生话辨析亮暗、鲜灰、慢慢变化。
  • 做三格颜色小样，避免颜色跳太快。
  • 把渐变规律放进色条、图形或小作品。
  • 用自查和同伴建议完成一处微修订。

课堂环节

约5分钟

1. 看见渐变

教师做什么：出示生活图和教材图，引导学生指出颜色从哪里变到哪里。

学生做什么：用“从___到___”说出一处颜色慢慢变化。

看什么证据：能指出起点、终点和变化方向。

约7分钟

2. 分清亮暗与鲜灰

教师做什么：用色条或教材图对比亮暗变化、鲜灰变化。

学生做什么：判断一组颜色是在变亮/变暗，还是变鲜/变灰。

看什么证据：能用儿童化语言说出亮暗或鲜灰。

约8分钟

3. 三格试色

教师做什么：示范三格颜色慢慢变化，提醒每次只变一点点。

学生做什么：完成一组三格或五格渐变小样。

看什么证据：小样中有连续变化，不是突然跳色。

约14分钟

4. 放进作品

教师做什么：提示学生把试色规律放进色条、图形或小作品。

学生做什么：完成一个带有渐变方向的小作品或局部设计。

看什么证据：作品中能看出明确的颜色变化方向。

约6分钟

5. 自查与微修订

教师做什么：组织学生按自查项看一看，并选择一处微修订。

学生做什么：说清自己的颜色怎么变，并根据建议改一处。

看什么证据：能表达变化，并完成一处可见修改。
