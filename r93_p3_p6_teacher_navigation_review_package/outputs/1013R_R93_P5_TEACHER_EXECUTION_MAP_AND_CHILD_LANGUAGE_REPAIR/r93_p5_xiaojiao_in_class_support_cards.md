# 小教课中支持卡

小教只做提示、追问、误区提醒和证据记录建议；本轮不接真实 UI。

## 1. 看见渐变
- 小教提醒：先让学生指出起点和终点，不急着讲术语。
- 小教追问：它是一下子跳过去，还是一层一层变过去？
- 小教误区提醒：学生可能把颜色多当成渐变。
- 小教证据记录建议：记录能用“从___到___”表达变化方向的学生。

## 2. 分清亮暗与鲜灰
- 小教提醒：先让学生分亮暗/鲜灰，再补明度/纯度。
- 小教追问：你看的是亮不亮，还是鲜不鲜？
- 小教误区提醒：学生可能把变浅和变灰混为一谈。
- 小教证据记录建议：记录能分类并说明理由的学生。

## 3. 三格试色
- 小教提醒：先做三格小样，不急着画完整作品。
- 小教追问：第二格是不是一下子跑太远了？
- 小教误区提醒：学生可能把三种不同颜色并排当成渐变。
- 小教证据记录建议：保存三格小样作为过程证据。

## 4. 放进作品
- 小教提醒：提醒教师分层发布任务，避免所有学生做复杂作品。
- 小教追问：别人一眼能看出你的颜色往哪里变吗？
- 小教误区提醒：学生可能追求复杂图案，忽略渐变规律。
- 小教证据记录建议：记录作品中能看出明确变化方向的学生样本。

## 5. 自查与微修订
- 小教提醒：评价只抓一处微修，不做大评比。
- 小教追问：改完以后，颜色变化更清楚了吗？
- 小教误区提醒：学生可能以为评价就是重画一遍。
- 小教证据记录建议：记录修改前后或学习单自查勾选。
