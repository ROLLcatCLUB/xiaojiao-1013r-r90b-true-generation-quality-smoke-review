# 三年级儿童化教师话术库

规则：短句、具体、可指向，先动作后概念。

## 1. 看见渐变

### 学生听得懂的话
- 颜色从哪里开始？
- 慢慢变到哪里？
- 用手指滑一滑。

### 可直接说出口的话术
- 先别急着说颜色名字，看看颜色是从哪里慢慢变到哪里的。
- 请你用手指滑一滑，颜色从哪里开始，滑到哪里结束？
- 你可以这样说：我看到颜色从___变到___。
- 它是一下子跳过去，还是中间一层一层慢慢变？
- 这种颜色慢慢变过去，美术里可以叫渐变。今天我们就研究它。

### 教师补充专业词
- 这种颜色一层一层慢慢变化，在美术里可以叫渐变。

### 教案书面表达
- 引导学生观察生活和教材图中的色彩连续变化，初步建立渐变感知。

## 2. 分清亮暗与鲜灰

### 学生听得懂的话
- 越来越亮
- 越来越暗
- 越来越鲜
- 越来越灰

### 可直接说出口的话术
- 这一页只看两件事：亮不亮，鲜不鲜。
- 这一组是越来越亮，还是越来越暗？
- 这一组是越来越鲜艳，还是越来越灰？
- 这张卡更像亮暗变化，还是鲜灰变化？放到对应篮子里。
- 美术里，亮不亮叫明度；鲜不鲜和纯度有关。我们先会看、会说就很好。

### 教师补充专业词
- 亮不亮叫明度，鲜不鲜和纯度有关。

### 教案书面表达
- 引导学生用儿童化语言辨析明度与纯度变化，降低概念负荷。

## 3. 三格试色

### 学生听得懂的话
- 让颜色排好队
- 每次只变一点点
- 第二格像第一格的朋友

### 可直接说出口的话术
- 今天我们先不画大作品，先让颜色排好队。
- 第一格先放一个你选好的颜色，别涂太厚，也别着急。
- 第二格不要跑太远，它要像第一格的朋友，只是亮一点或灰一点。
- 第三格继续慢慢变，不要一下子跳过去。
- 你看这一组第二格一下子跳太远了，像不像中间少了一步？
- 现在你也试一条三格路，只要三格，先别急着画完整作品。
- 我先看三件事：从哪里开始，往哪里变，中间有没有跳太快。

### 教师补充专业词
- 这种一格一格有规律的变化，可以形成明度或纯度渐变。

### 教案书面表达
- 通过三格小样训练学生控制渐变幅度，形成可迁移的色彩操作经验。

## 4. 放进作品

### 学生听得懂的话
- 把颜色小路放进作品
- 别人能看出方向
- 不用画很复杂

### 可直接说出口的话术
- 你可以选适合自己的任务：色条、图形，或者小作品。今天最重要的是看得出颜色怎么变。
- 先别急着涂。你要先想：颜色从哪里开始，往哪里走？
- 刚才的小样不是结束，它是一条颜色小路。现在把这条小路放进你的图形里。
- 现在给你一段安静时间。记住：不用画很多，先把一处渐变做清楚。
- 我来看看你的颜色从哪里开始，往哪里变，中间是不是太突然。
- 同桌只帮看一件事：你能不能看出颜色往哪里变？
- 我们先看几处颜色变化，不比谁画得最多，只看哪里变得清楚。

### 教师补充专业词
- 把渐变规律迁移到画面局部，形成有方向、有层次的色彩表现。

### 教案书面表达
- 引导学生将试色规律迁移到图形或小作品中，形成可见的渐变表达。

## 5. 自查与微修订

### 学生听得懂的话
- 看一看
- 说一说
- 只改一处

### 可直接说出口的话术
- 最后我们不急着评谁最好，先看自己有没有做到三件事。
- 同桌只说一处：我看到___，我建议___。
- 你只需要改一处，不用重画。让颜色变化更清楚就可以。
- 改的时候想一想：它现在是不是更像慢慢变了？
- 你可以这样说：我改了___，因为原来___，现在___。

### 教师补充专业词
- 通过自评、同伴建议和微修订形成作品改进证据。

### 教案书面表达
- 组织学生依据评价要点进行自查、表达和局部修订，强化学习证据。
