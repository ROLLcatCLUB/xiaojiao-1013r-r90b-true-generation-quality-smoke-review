《色彩的渐变》教师课堂执行地图

从“教学过程说明书”推进到“第一步、第二步、第三步”的课堂执行手册。

1. 看见渐变2. 分清亮暗与鲜灰3. 三格试色4. 放进作品5. 自查与微修订

0. 执行原则

  • 教师先看到课堂动作，再展开支架和意图。
  • 每一步都要有学生动作和大屏或材料状态。
  • 先用儿童语言，再由教师轻轻补专业词。
  • 小教支持嵌入步骤，不作为旁白堆在最后。

观察比较型

1. 看见渐变

学生话：颜色从哪里开始？ / 慢慢变到哪里？ / 用手指滑一滑。

教师补专业词：这种颜色一层一层慢慢变化，在美术里可以叫渐变。

1.1 打开图片，先静静看

大屏 / 支架 / 小教 / 证据

大屏 / 材料：大屏显示三张图，每张图只保留一个箭头。

预期回应：学生能指出天空、山峦或花朵中的一处变化。

学生支架：不会说时先用手指，不要求完整句。

小教支持：提醒教师先看方向，不讲概念。

证据检查：学生眼睛能跟随箭头看变化。

卡住时：把图片减少到一张最明显的图。

师打开生活渐变图或教材图，给学生5秒安静观察。

话先别急着说颜色名字，看看颜色是从哪里慢慢变到哪里的。

生安静看图，找一处颜色慢慢变化。

进多数学生能找到一处变化后进入下一步。

1.2 用手指出方向

大屏 / 支架 / 小教 / 证据

大屏 / 材料：大屏保留箭头，教师可圈起点和终点。

预期回应：从深色滑到浅色；从蓝色滑到粉色。

学生支架：给起点贴一个小圆点，终点贴一个小旗。

小教支持：如果学生只说好看，提示追问从哪里到哪里。

证据检查：能指出起点和终点。

卡住时：教师示范一次手指滑动。

师请学生用手指从起点滑到终点。

话请你用手指滑一滑，颜色从哪里开始，滑到哪里结束？

生用手指比划颜色变化方向。

进至少2名学生能指出方向后进入表达。

1.3 说一句从哪里到哪里

大屏 / 支架 / 小教 / 证据

大屏 / 材料：大屏显示句式：从___到___。

预期回应：我看到颜色从蓝色变到粉色。

学生支架：句式板书保留在屏幕边角。

小教支持：给教师提示：不要纠结颜色名称是否精确。

证据检查：学生能说出完整或半完整句。

卡住时：让同桌先说，再请一名学生说。

师让学生套用句式说一句。

话你可以这样说：我看到颜色从___变到___。

生用句式说出一处变化。

进有学生能说出变化方向后进入比较。

1.4 判断是不是慢慢变

大屏 / 支架 / 小教 / 证据

大屏 / 材料：大屏放大一处连续色带。

预期回应：它是一层一层变过去的。

学生支架：用“一层一层”替代“连续”术语。

小教支持：提醒教师把“颜色多”和“慢慢变”分开。

证据检查：学生能说出慢慢变的理由。

卡住时：对比一张颜色跳得很快的例子。

师教师追问变化是否连续。

话它是一下子跳过去，还是中间一层一层慢慢变？

生判断“慢慢变”还是“跳过去”。

进学生能说慢慢变后进入概念桥接。

1.5 轻轻引出渐变

大屏 / 支架 / 小教 / 证据

大屏 / 材料：大屏出现课题：色彩的渐变。

预期回应：渐变就是颜色慢慢变。

学生支架：课题旁写：渐变=慢慢变。

小教支持：提醒教师术语只出现一次，不展开理论。

证据检查：学生能把渐变和慢慢变联系起来。

卡住时：若学生记不住术语，保留慢慢变即可。

师教师用学生语言引出“渐变”。

话这种颜色慢慢变过去，美术里可以叫渐变。今天我们就研究它。

生读课题，回到“颜色慢慢变过去”。

进学生理解课堂主题后进入亮暗/鲜灰辨析。

概念辨析型

2. 分清亮暗与鲜灰

学生话：越来越亮 / 越来越暗 / 越来越鲜 / 越来越灰

教师补专业词：亮不亮叫明度，鲜不鲜和纯度有关。

2.1 出示两组色条

大屏 / 支架 / 小教 / 证据

大屏 / 材料：大屏左右两组色条，文字为亮暗/鲜灰。

预期回应：左边越来越亮，右边越来越灰。

学生支架：用左右分区降低判断难度。

小教支持：提醒教师不要先讲定义。

证据检查：学生能看出两组不同。

卡住时：只保留一组色条先看亮暗。

师左边放深浅色条，右边放鲜灰色条。

话这一页只看两件事：亮不亮，鲜不鲜。

生观察左右两组色条。

进学生能说左右不同后进入分类。

2.2 先判断亮暗

大屏 / 支架 / 小教 / 证据

大屏 / 材料：左侧色条放大。

预期回应：越来越亮/越来越暗。

学生支架：给学生两个选项，不开放太多词。

小教支持：小教提示教师让学生先二选一。

证据检查：学生能用亮/暗判断。

卡住时：用黑白深浅卡替代彩色色条。

师指左侧色条，只问亮暗。

话这一组是越来越亮，还是越来越暗？

生举手或口头判断亮暗。

进大多数学生能判断后看鲜灰。

2.3 再判断鲜灰

大屏 / 支架 / 小教 / 证据

大屏 / 材料：右侧色条放大。

预期回应：越来越灰；没有那么鲜了。

学生支架：允许学生说“不鲜了”。

小教支持：提示教师接纳儿童话。

证据检查：学生能说鲜或灰。

卡住时：拿鲜艳色和灰色实物对比。

师指右侧色条，只问鲜灰。

话这一组是越来越鲜艳，还是越来越灰？

生判断鲜灰变化。

进学生能判断后进入分类活动。

2.4 把色卡放进篮子

大屏 / 支架 / 小教 / 证据

大屏 / 材料：大屏显示两个篮子：亮暗、鲜灰。

预期回应：这张是亮暗；这张是鲜灰。

学生支架：同桌先讨论再放。

小教支持：小教提醒教师追问选择理由。

证据检查：学生能分类并说出理由。

卡住时：不会分类时先问亮不亮，再问鲜不鲜。

师请学生把色卡放到亮暗篮子或鲜灰篮子。

话这张卡更像亮暗变化，还是鲜灰变化？放到对应篮子里。

生拖动/贴卡/口头选择类别。

进完成2-3张分类后桥接专业词。

2.5 补专业词并收束

大屏 / 支架 / 小教 / 证据

大屏 / 材料：板书：明度=亮暗；纯度=鲜灰。

预期回应：明度是亮暗，纯度和鲜灰有关。

学生支架：只保留等号式板书。

小教支持：提醒教师不要扩展色相等额外概念。

证据检查：学生能把专业词和儿童话连起来。

卡住时：如果学生负担大，只保留亮暗/鲜灰。

师教师将儿童话对应到美术词。

话美术里，亮不亮叫明度；鲜不鲜和纯度有关。我们先会看、会说就很好。

生跟读关键词，不背定义。

进学生会看会说后进入三格试色。

示范试做型

3. 三格试色

学生话：让颜色排好队 / 每次只变一点点 / 第二格像第一格的朋友

教师补专业词：这种一格一格有规律的变化，可以形成明度或纯度渐变。

3.1 拿出三格小样纸

大屏 / 支架 / 小教 / 证据

大屏 / 材料：大屏显示三格空框和箭头。

预期回应：知道先做小样。

学生支架：三格框已印好，减少画框负担。

小教支持：提醒教师先控任务范围。

证据检查：学生找到三格位置。

卡住时：教师帮学生指认第1、2、3格。

师展示三格空框，说明今天先试一条小路。

话今天我们先不画大作品，先让颜色排好队。

生拿出三格小样纸或在学习单上找到三格。

进全班找到三格后开始第一格。

3.2 画第一格

大屏 / 支架 / 小教 / 证据

大屏 / 材料：第1格被涂上原色。

预期回应：第一格颜色比较稳定。

学生支架：可用彩铅/油画棒降低水分风险。

小教支持：提示教师关注材料控制。

证据检查：学生完成第一格。

卡住时：材料慢的学生可直接用色卡。

师示范把选好的颜色放入第一格。

话第一格先放一个你选好的颜色，别涂太厚，也别着急。

生在第一格涂色。

进大多数学生有第一格后进入第二格。

3.3 画第二格

大屏 / 支架 / 小教 / 证据

大屏 / 材料：第2格显示轻微变化，旁边写只变一点点。

预期回应：第二格比第一格略亮/略灰。

学生支架：给学生一句判断：像不像第一格的朋友？

小教支持：小教提醒教师看第二格是否跳太远。

证据检查：第二格与第一格关系接近。

卡住时：让学生把第二格调回接近第一格。

师示范第二格只做轻微变化。

话第二格不要跑太远，它要像第一格的朋友，只是亮一点或灰一点。

生调出或选出第二格颜色。

进第二格基本完成后进入第三格。

3.4 画第三格

大屏 / 支架 / 小教 / 证据

大屏 / 材料：第3格显示继续变化，箭头连起三格。

预期回应：三格能看出方向。

学生支架：保留箭头提醒变化方向。

小教支持：提示教师要求方向一致。

证据检查：学生三格有方向。

卡住时：让学生先说方向再改色。

师示范第三格继续沿同方向变化。

话第三格继续慢慢变，不要一下子跳过去。

生完成第三格。

进多数学生完成三格后看反例。

3.5 对比跳色反例

大屏 / 支架 / 小教 / 证据

大屏 / 材料：大屏显示自然三格和跳色三格对比。

预期回应：左边更自然，右边跳太快。

学生支架：用“少了一步”解释断层。

小教支持：小教提示教师别批评学生，只说可以补一步。

证据检查：学生能识别跳色。

卡住时：让学生找哪一格需要靠近前一格。

师展示一组中间跳太快的三格。

话你看这一组第二格一下子跳太远了，像不像中间少了一步？

生判断哪一组更自然。

进学生能说出问题后进入自查。

3.6 学生试做自己的三格

大屏 / 支架 / 小教 / 证据

大屏 / 材料：大屏保留步骤：选颜色、三格、箭头。

预期回应：我做的是从___到___。

学生支架：基础做三格，进阶做五格。

小教支持：小教提示教师巡视起点、方向、中间格。

证据检查：学生有一组三格小样。

卡住时：卡住学生可用色卡排一排。

师让学生独立完成一条三格路。

话现在你也试一条三格路，只要三格，先别急着画完整作品。

生完成自己的三格小样。

进学生完成小样后进入巡视判断。

3.7 巡视并点拨

大屏 / 支架 / 小教 / 证据

大屏 / 材料：大屏显示三点自查。

预期回应：能说起点、方向、中间格。

学生支架：用手指沿三格滑动检查。

小教支持：小教记录典型问题和可展示样本。

证据检查：教师能看到过程证据。

卡住时：时间不足时只保留三格和一句话。

师教师巡视，只看三个点：起点、方向、中间是否跳。

话我先看三件事：从哪里开始，往哪里变，中间有没有跳太快。

生按三点检查自己的小样。

进有可用小样后进入作品迁移。

创作实践型

4. 放进作品

学生话：把颜色小路放进作品 / 别人能看出方向 / 不用画很复杂

教师补专业词：把渐变规律迁移到画面局部，形成有方向、有层次的色彩表现。

4.1 选择任务层级

大屏 / 支架 / 小教 / 证据

大屏 / 材料：大屏显示基础/进阶/挑战三栏。

预期回应：我选基础/进阶/挑战。

学生支架：允许学生从基础做起。

小教支持：小教提醒教师不要让复杂图案抢走目标。

证据检查：学生能选定任务。

卡住时：选择困难时统一做基础色条。

师发布基础、进阶、挑战三类任务。

话你可以选适合自己的任务：色条、图形，或者小作品。今天最重要的是看得出颜色怎么变。

生选择一个任务层级。

进学生选定任务后标方向。

4.2 先标变化方向

大屏 / 支架 / 小教 / 证据

大屏 / 材料：大屏显示起点、箭头、终点。

预期回应：从左到右；从里到外；从上到下。

学生支架：用箭头提示方向。

小教支持：小教提示教师先问方向再看颜色。

证据检查：学生能说出方向。

卡住时：让学生用手指在画面上滑一次。

师要求学生先画箭头或心里确定方向。

话先别急着涂。你要先想：颜色从哪里开始，往哪里走？

生在纸上轻轻标方向或口头说方向。

进方向明确后迁移小样。

4.3 把三格小样放进图形

大屏 / 支架 / 小教 / 证据

大屏 / 材料：大屏显示小样到图形的迁移箭头。

预期回应：我把渐变放在花瓣/山/圆形里。

学生支架：先放一小块，不要求整张画满。

小教支持：小教提醒教师看是否迁移了规律。

证据检查：作品中出现渐变局部。

卡住时：学生不会迁移时让他照着三格画一条色带。

师示范把小样规律放到一个形状里。

话刚才的小样不是结束，它是一条颜色小路。现在把这条小路放进你的图形里。

生把三格规律迁移到作品局部。

进能迁移后进入创作。

4.4 开始创作

大屏 / 支架 / 小教 / 证据

大屏 / 材料：大屏保留三点提醒：起点、方向、慢慢变。

预期回应：作品中有一处颜色慢慢变化。

学生支架：大屏提醒替代教师反复口头提醒。

小教支持：小教提示教师减少打断，保留巡视。

证据检查：学生进入持续创作。

卡住时：慢学生只完成局部即可。

师给学生安静创作时间，教师不频繁打断。

话现在给你一段安静时间。记住：不用画很多，先把一处渐变做清楚。

生完成作品或局部设计。

进创作进行一段后巡视点拨。

4.5 巡视抓三个点

大屏 / 支架 / 小教 / 证据

大屏 / 材料：大屏显示巡视三点。

预期回应：从这里开始，往这里变。

学生支架：让学生用手指解释作品。

小教支持：小教生成个别追问：中间是不是跳太快？

证据检查：教师获得过程判断。

卡住时：如果方向不清，让学生加箭头或补一格。

师巡视时只看起点、方向、过渡。

话我来看看你的颜色从哪里开始，往哪里变，中间是不是太突然。

生向教师说明自己的变化方向。

进大部分作品有方向后同伴快看。

4.6 同桌快看

大屏 / 支架 / 小教 / 证据

大屏 / 材料：大屏显示同桌提示句。

预期回应：我看到它从___变到___。

学生支架：同伴只提一个点，避免干扰创作。

小教支持：小教提醒教师控制互评时间。

证据检查：同伴能指出或指出不清。

卡住时：看不出时让作者补一处过渡。

师安排同桌只看一件事：方向清不清。

话同桌只帮看一件事：你能不能看出颜色往哪里变？

生同桌互看并指出方向。

进同伴反馈后进入收束。

4.7 收束到可展示样本

大屏 / 支架 / 小教 / 证据

大屏 / 材料：大屏或实物投影展示局部。

预期回应：这一处从___变到___。

学生支架：只展示局部，减少比较压力。

小教支持：小教记录可作为评价样本的作品。

证据检查：学生能从作品中看出渐变。

卡住时：时间不足时跳过全班展示，直接自查。

师选2-3个样本，不做大范围展示。

话我们先看几处颜色变化，不比谁画得最多，只看哪里变得清楚。

生看样本，说出变化方向。

进选好样本后进入自查微修订。

展示修订型

5. 自查与微修订

学生话：看一看 / 说一说 / 只改一处

教师补专业词：通过自评、同伴建议和微修订形成作品改进证据。

5.1 打开三项自查

大屏 / 支架 / 小教 / 证据

大屏 / 材料：大屏显示：方向、慢慢变、能说清。

预期回应：能找到一处渐变。

学生支架：自查项不超过三条。

小教支持：小教提醒教师压缩评价维度。

证据检查：学生能进入自查。

卡住时：教师带全班一起读三问。

师出示自查三问。

话最后我们不急着评谁最好，先看自己有没有做到三件事。

生看自己的小样或作品。

进学生知道查什么后进入同伴建议。

5.2 同伴只提一处建议

大屏 / 支架 / 小教 / 证据

大屏 / 材料：大屏显示同伴句式。

预期回应：我看到这里变得清楚，我建议中间再浅一点。

学生支架：句式帮助学生避免泛泛说好看。

小教支持：小教提示教师控制为一条建议。

证据检查：学生能获得一条可改建议。

卡住时：如果不会说，只指出看不清的位置。

师安排同伴用固定句式给建议。

话同桌只说一处：我看到___，我建议___。

生给同伴一条建议。

进每人收到一条建议后微修。

5.3 选择一处微修

大屏 / 支架 / 小教 / 证据

大屏 / 材料：大屏显示只改一处。

预期回应：我改第二格/中间过渡/方向不清的地方。

学生支架：用圈画标出要改的位置。

小教支持：小教提醒教师防止学生重画。

证据检查：学生明确修改点。

卡住时：时间不足只口头说明要改哪里。

师要求学生只改一处。

话你只需要改一处，不用重画。让颜色变化更清楚就可以。

生选择一处修改。

进学生确定修改点后动手。

5.4 完成小修改

大屏 / 支架 / 小教 / 证据

大屏 / 材料：大屏保留自查三问。

预期回应：颜色过渡更自然。

学生支架：允许很小的修改。

小教支持：小教记录典型微修案例。

证据检查：有可见修改或口头修改方案。

卡住时：材料不便时用铅笔箭头或文字说明替代。

师给1-2分钟完成局部修改。

话改的时候想一想：它现在是不是更像慢慢变了？

生完成局部修订。

进完成后说一说。

5.5 说清修改并收课

大屏 / 支架 / 小教 / 证据

大屏 / 材料：大屏显示表达句式和下节课提示。

预期回应：我改了中间一格，因为原来跳太快，现在更慢慢变了。

学生支架：用句式保护表达困难学生。

小教支持：小教记录能说清修改的学生。

证据检查：学生能说明修改与渐变有关。

卡住时：不会说完整句时只说改哪里。

师请学生说一处修改，教师连接下一课。

话你可以这样说：我改了___，因为原来___，现在___。

生说出修改原因。

进完成表达后收束到下节课“渐变的节奏”。
