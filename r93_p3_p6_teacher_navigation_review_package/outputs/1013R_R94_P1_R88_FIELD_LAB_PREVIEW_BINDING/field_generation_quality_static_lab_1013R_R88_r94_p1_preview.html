底层字段与内容生成质量验证样板

这是一页独立静态页，用来验证“大单元主线 → 单课切片 → 字段内部结构 → 派生物”的生成质量。页面只标注外显字段、工程字段、模型生成内容占位，不展示提示词，不调用模型，不写入正式备课本。

一、生成层级

unit_spine 大单元主线

先确定单元主题、大观念、基本问题和任务证据，不直接从空白写单课。

lesson_slice 单课切片

把大单元主线切到当前课时，形成本课依据、目标、任务和证据。

field_internal 字段内部结构

每个字段内部继续拆成可观察、可上课、可评价的子结构。

derived_objects 派生物

大屏、学习单、评价表只从已确认的教学过程和证据派生。

课件故事板结构

页序页面标题屏幕文字学生动作
第1页封面：色彩的渐变色彩的渐变 颜色慢慢变过去读课题，说一说屏幕上的颜色有没有慢慢变化。
第2页生活中的渐变找一找：颜色从哪里变到哪里？用“从___到___”说出一处生活或教材图中的渐变。
第3页教材图观察：明度与纯度亮一点 / 暗一点 鲜一点 / 灰一点判断一组颜色是在变亮/变暗，还是变鲜/变灰。
第4页核心词：亮暗 / 鲜灰 / 慢慢变化亮暗：明度 鲜灰：纯度 慢慢变化：渐变把教师出示的色卡贴到“亮暗”或“鲜灰”类别。
第5页调色游戏每次只变一点点选择一条路线，做三格颜色慢慢变化的小样。
第6页教师示范选颜色 -> 做小样 -> 标方向 -> 放进作品观察示范，并在学习单上勾选自己要做亮暗还是鲜灰变化。
第7页学生创作任务基础：三格 进阶：五格 挑战：放进小作品选择一个层级完成渐变小样或小作品。
第8页作品自查我看得出起点 我看得出方向 中间没有跳太快 我能说一句用自查句检查作品，圈出一处可调整地方。
第9页展示交流我看到... 我觉得... 我建议...说出同伴作品的一处优点和一处建议。
第10页微修订只改一处，让颜色更自然根据自查或同伴建议微修订一处。

一页学生版学习单

  • 找一找: 我看到颜色从___慢慢变到___。
  • 试一试: 选一种颜色，画三格，让它慢慢变。
  • 查一查: 我能说出颜色从哪里变到哪里。

边界 一页可打印草案，但不是打印定稿。

评价拆分

教师观察版

  • 看得见：能发现渐变
  • 试得出：能做出连续变化
  • 用得上：能把渐变用到作品里
  • 说得清：能说明颜色怎么变化
  • 改得动：能根据建议修一处

学生自评版

  • 我看见颜色慢慢变了
  • 我试出了三层颜色变化
  • 我能说出颜色从哪里变到哪里
