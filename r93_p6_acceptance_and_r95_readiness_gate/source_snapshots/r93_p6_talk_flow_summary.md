# R93-P6 本课教师话术总览

这些话术来自 R93-P5 execution map，本轮只做阅读重组，不新增正式材料。

## 1. 看见渐变
- 先别急着说颜色名字，看看颜色是从哪里慢慢变到哪里的。
- 请你用手指滑一滑，颜色从哪里开始，滑到哪里结束？

## 2. 分清亮暗与鲜灰
- 这一页只看两件事：亮不亮，鲜不鲜。
- 美术里，亮不亮叫明度；鲜不鲜和纯度有关。我们先会看、会说就很好。

## 3. 三格试色
- 第二格不要跑太远，它要像第一格的朋友，只是亮一点或灰一点。
- 现在你也试一条三格路，只要三格，先别急着画完整作品。

## 4. 放进作品
- 你可以选适合自己的任务：色条、图形，或者小作品。今天最重要的是看得出颜色怎么变。
- 现在给你一段安静时间。记住：不用画很多，先把一处渐变做清楚。

## 5. 自查与微修订
- 最后我们不急着评谁最好，先看自己有没有做到三件事。
- 你只需要改一处，不用重画。让颜色变化更清楚就可以。
